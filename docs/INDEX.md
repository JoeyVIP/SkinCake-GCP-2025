## 📝 文檔維護

### 更新頻率
- **CHANGELOG.md**: 每次版本發布
- **故障排除**: 發現新問題時更新
- **API 文檔**: API 變更時更新
- **其他文檔**: 根據需要更新

### 貢獻文檔
1. Fork 專案
2. 創建文檔分支 `docs/your-update`
3. 提交 Pull Request
4. 等待審核合併
