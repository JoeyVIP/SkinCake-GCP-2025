## ✨ 專案特色

- 🚀 **現代化架構** - Next.js 14 + TypeScript + Tailwind CSS
- 📱 **響應式設計** - 完美支援各種裝置
- 🔍 **SEO 優化** - SSG/ISR 混合渲染
- 🎨 **韓系美學** - 清新粉色主題設計
- 📰 **動態內容** - WordPress CMS 整合

## 🚀 快速開始

```bash
# 克隆專案
git clone https://github.com/JoeyVIP/SkinCake-GCP-2025.git
cd SkinCake-GCP-2025

# 安裝依賴
npm install

# 設置環境變數
cp .env.example .env.local

# 啟動開發伺服器
npm run dev
```

訪問 http://localhost:3000 查看網站
