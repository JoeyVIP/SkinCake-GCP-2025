## 🎯 **第一階段：最後衝刺** (2025/07/15-07/17)

### 🚨 **高優先級任務** (2025/07/16 前完成)

#### 1. CategoryPageClient 組件修復
- **問題**: 模組導入錯誤 `Can't resolve './CategoryPageClient'`
- **影響**: 分類頁面無法正常運行
- **解決方案**: 
  - 修復組件導入路徑
  - 確保 TypeScript 類型正確
  - 測試所有分類頁面功能
- **工時估算**: 4小時

#### 2. 圖片載入問題修復
- **問題**: 
  - `bn01.jpg`, `bn02.jpg`, `bn03.jpg` (輪播圖) 404錯誤
  - `choice01-06.jpg` (分類卡片) 404錯誤
- **影響**: 首頁視覺效果缺失
- **解決方案**:
  - 確認圖片文件存在於 `public/images/`
  - 檢查 Next.js 圖片路徑配置
  - 測試所有圖片載入
- **工時估算**: 2小時

#### 3. WordPress API 快取優化
- **問題**: `Failed to set Next.js data cache, items over 2MB`
- **影響**: 建置時間過長，可能影響部署
- **解決方案**:
  - 減少 `getAllPosts` 的 `per_page` 參數
  - 移除不必要的 `_embed` 數據
  - 實施分頁載入策略
- **工時估算**: 3小時
